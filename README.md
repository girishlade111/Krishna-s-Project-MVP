## ‚ÑπÔ∏è About This Project

This project is a front-end exercise. The primary goal was to take a static image of a live website and reproduce its design and layout using core web technologies. It demonstrates skills in:

* **HTML5:** Semantic structure and organization of content.
* **CSS3:** Advanced layout (Grid & Flexbox), custom properties (variables), and precise styling for a "pixel-perfect" result.
* **JavaScript:** Minimal, non-obtrusive JS for potential micro-interactions (like the video play button).

This is **not** a functional website; it has no backend, and all content is hardcoded. It serves as a portfolio piece showcasing attention to detail and front-end capabilities.

---

## ‚ú® Features Implemented

This clone includes the following sections from the original homepage:

* **Header:** Dual-layer navigation with a top utility bar and a main navigation bar.
* **Hero Section:** Full-width hero with a background image, text overlay, and call-to-action.
* **Featured News:** A 4-column grid of news articles with images and categories.
* **TUM in Numbers:** A 4-column "facts" section with colored accent cards.
* **Student Life Section:** A 3-column layout featuring text, an image, and a "Prospective Students" callout box.
* **Meet TUM Section:** A similar 3-column layout with an image, text, and a "Latest Insights" card.
* **Alumni Section:** A simple section highlighting two key statistics in large cards.
* **Campus Guide:** A 2-column section featuring a video placeholder and related links.
* **Testimonial:** A full-width banner with a quote and an accompanying image.
* **People of TUM:** A 6-column grid showcasing individuals at the university.
* **Call-to-Action:** A 2-card CTA section for "Contact" and "Press."
* **Footer:** A comprehensive 5-column footer with links, contact info, and social media.
